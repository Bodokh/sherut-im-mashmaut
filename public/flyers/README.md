# A4 flyer directions

Open `/flyers/index.html?direction=1` through `/flyers/index.html?direction=5`.

Each direction prints as a full-bleed 210 × 297 mm A4 page. The internal design
canvas preserves high-precision coordinates, while both screen preview and
print output scale it to exact A4 dimensions.

The QR code opens the canonical WhatsApp contact at `+972 52-757-5290`.

For final press output, replace the current 960 × 540 hero image in directions
that use it with a high-resolution original, then export the selected page to
PDF with background graphics enabled.
